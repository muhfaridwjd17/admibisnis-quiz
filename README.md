# 🎓 AdminBiz - Portal Kuis Administrasi Bisnis

Portal kuis interaktif modern untuk mahasiswa D4 Administrasi Bisnis PNUP.

## Fitur
- 10 Mata Kuliah, masing-masing 10 soal
- Timer otomatis 10 menit
- Analitik nilai & progress tracker
- Leaderboard personal
- Ringkasan materi pembelajaran
- Dark mode modern (glassmorphism design)
- Responsive untuk mobile & desktop
- Data tersimpan di browser (localStorage)

## Mata Kuliah
1. Pendidikan Kewarganegaraan
2. Manajemen Portofolio & Investasi
3. Perilaku Konsumen
4. Kepemimpinan
5. Pelayanan Prima
6. English for Business Correspondent
7. Simulasi Bisnis
8. Aplikasi Komputer Bisnis
9. Kebijakan Bisnis
10. Sistem Informasi Manajemen

## Deploy ke GitHub Pages
1. Push semua file ke repository GitHub
2. Settings → Pages → Branch: main → Folder: / (root)
3. Akses via: `https://[username].github.io/[repo-name]`
